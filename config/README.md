# Config

PAMS는 "설정 파일 기반" 원칙에 따라 투자 판단에 관련된 모든 규칙을 코드가 아닌 파일로 관리한다.
코드는 이 파일들을 읽어 실행할 뿐, 규칙 자체를 하드코딩하지 않는다.

| 디렉터리 | 내용 | 담당 Phase |
|---|---|---|
| `app/` | 애플리케이션 전역 설정(로그 레벨, 기준통화, 타임존 등) | Phase 1 |
| `ips/` | 투자헌장(Investment Policy Statement) - 목표 비중, 손절/익절, 리밸런싱 주기 등 | Phase 3 |
| `rules/` | Rule Engine이 평가하는 개별 규칙 정의 (추가/삭제 가능) | Phase 3 |

`ips/`, `rules/`의 실제 스키마와 예시 파일은 Phase 3(IPS 엔진)에서 확정한다.
