오늘의 액션 — 언제 사고/팔지

모든 신호는 Rule/트리거 엔진의 계산 결과다. 실제 매매 실행 여부는 사용자가 결정한다.

거래 입력 (매수 · 매도 · 입출금)

거래를 기록하면 보유수량·평단가·예수금·비중이 자동으로 다시 계산된다.

주식 종목 (보유 · 매수/매도 트리거)

＋ 매수/익절/손절 가격선(트리거) 설정

매수선↓ 매수 · 익절선↑ 매도 · 손절선↓ 매도. 현재가가 선을 건드리면 '오늘의 액션'에 신호가 뜬다. (순서: 손절 < 매수 < 익절)

＋ 새 종목 추가

추가하면 거래 입력·시세 자동수집에서 이 종목을 쓸 수 있다.
