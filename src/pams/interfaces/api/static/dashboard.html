가격 트리거 설정

매수/익절/손절 가격선(트리거) 설정

매수선↓ 매수 · 익절선↑ 매도 · 손절선↓ 매도. 현재가가 선을 건드리면 '오늘의 액션'에 신호가 뜬다. (순서: 손절 < 매수 < 익절)

＋ 비율 기준 트리거 일괄 설정 (보유 종목 전체)

손절선 = 평단가×(1−%), 익절선 = 평단가×(1+%), 매수선 = 현재가×(1−%). 이미 평단가 대비 크게 벗어나 순서(손절<매수<익절)가 깨지는 종목은 건너뛰고 아래에 표시한다.

5개 지표를 다수결(동률이면 더 위험한 쪽 채택)로 A(강세)~E(위기) 등급을 매긴다. 3개 미만 입력되면 "판단 보류"로 응답한다(임의 등급 부여 금지). 매수 필수조건(5장)의 "시장 상태 C 이상" 판정 근거로 쓴다. VIX·KOSPI 등락률은 비워두면 자동조회한다 (실패하면 데이터 누락으로 처리— 명시 입력이 있으면 그대로 존중).

② 매수 필수조건 판정 (buy_rules.md B-1 — AND 게이트)

4개 조건을 모두 충족해야 매수 후보다. "종목분석" 탭·위 시장국면 판정에서 계산한 값이 자동으로 채워진다 — 그대로 쓰거나 직접 고쳐도 된다. 매수 전 B-3 체크리스트의 포트폴리오 집중도 한도(개별 20%, 초우량 예외 30%)는 이 4개 조건에는 포함되지 않는다 — "분석" 탭의 알림(규정준수 위반)에서 별도로 확인한다.

아직 판정 결과가 없다.

③ 매도 검토 (sell_rules.md 5장 — 자동집행 아님)

매도는 논리 훼손이 최우선이다. 아래 신호는 검토를 권고할 뿐이며, 실행 전 반드시 사용자가 직접 확인한다. DCF 괴리율은 "종목분석" 탭 결과에서 자동으로 채워진다.
