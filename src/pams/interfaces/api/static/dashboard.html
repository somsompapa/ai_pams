가격 트리거 설정

매수/익절/손절 가격선(트리거) 설정

매수선↓ 매수 · 익절선↑ 매도 · 손절선↓ 매도. 현재가가 선을 건드리면 '오늘의 액션'에 신호가 뜬다. (순서: 손절 < 매수 < 익절)

＋ 비율 기준 트리거 일괄 설정 (보유 종목 전체)

손절선 = 평단가×(1−%), 익절선 = 평단가×(1+%), 매수선 = 현재가×(1−%). 이미 평단가 대비 크게 벗어나 순서(손절<매수<익절)가 깨지는 종목은 건너뛰고 아래에 표시한다.

종목 관리 (추가·수정·삭제)

추가하면 거래 입력·시세 자동수집에서 이 종목을 쓸 수 있다.

종목 목표비중 설정 (Tier 2)

매수 트리거 = 목표−매수밴드(이하면 매수) · 매도 트리거 = 목표+매도밴드(이상이면 매도). 결과는 '포트폴리오' 탭의 종목별 목표비중 표에 반영된다.
